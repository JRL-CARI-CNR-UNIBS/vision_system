class vision_system.camera.Camera(color_image_topic: str = '/camera/color/image_raw', depth_image_topic: str = '/camera/depth/image_raw', camera_info_topic: str = '/camera/depth/camera_info', frames_approx_sync: bool = False, depth_frame_encoding: str = '32FC1')ÔÉÅ

Bases: Node

Camera class for acquiring images (color and depth) from an RGBD-camera.

This class is responsible for subscribing to the image and depth topics making it easy to access to camera frames, converting the images using cv_bridge, and optionally applying post-processing.
